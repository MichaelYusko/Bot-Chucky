Welcome to Bot-Chucky's documentation!
======================================

v0.0.3 (`Github`_)

Bot-Chucky is a Facebook Messenger Bot API, capable of interacting with the user,
providing integration with Twitter, Gmail, SoundCloud, StackExchange and weather updates.
